Instructions

1. Unzip the textFile.txt.zip

2. Extract all images from the CelebA dataset into the 'data' folder in this directory. 

3. Change num_images_to_sort in the readText.py file for how many images you want to sort.

4. Change attribute for which attribute you want to sort by.

5. Run the readText.py script.
